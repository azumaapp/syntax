Node api를 구글에서 검색 후, 다양한 모듈들의 api 문서를 참조해 코딩을 계속 진행하면 된다.

1. http를 들어가보자.
2. 그 안에서 createServer 함수를 찾아보자.
3. api 설명을 보면서 코드와 비교 대조 해본다. 
4. api안에서 함수의 파라미터에 []표시는 있을수도 있고 없을 수도 있다는 뜻이다.
5. 함수를 사용하는 형식들이 여러개가 있을 수 있는데 이런 각각의 형식들을 시그니쳐(signature)라고 한다.
6. 이러한 문서들은, 해당 언어(nodeJS)를 사용하여 application(앱을) programming(프로그래밍하는) interface(접점)이라 하여 api 문서라고 한다. (다른 언어 역시 api 문서를 가지고 있다.)
7. 실무에서는 이렇게 api 문서를 보면서 다른 모듈의 함수를 어떻게 사용하는지 보고, 시그니쳐 중 필요한 것을 취사 선택해 사용할 수 있다.

참조한 노드API : https://nodejs.org/api/n-api.html
