express 실전수업
1. 깃에서, 해당폴더 안에 있는 express-app.zip을 받아서 깐다.
2. 해당 파일 위치에서 터미널을 열고 npm i express, npm i, npm audit fix --force
3. 구글에 express api 검색 -> 첫번째 페이지 -> 시작하기 -> hello world 부분 복사
4. main.js 안의 기존코드 (nodejs 서버로 구현)을 주석화하고 복사한 부분 위에 복붙
5. pm2 start main.js --watch 로 실행
6. localhost:3000에서 Hello World! 출력 확인
7. nodejs 서버 -> express 로 구현환경 변경하기 시작
8. 다 하고 나서 pm2 kill 로 끄기
