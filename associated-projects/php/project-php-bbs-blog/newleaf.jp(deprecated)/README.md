# "NewLeaf" One Page Website
"NewLeaf" 홈페이지의 저작권은 다음의 개발자에게 있습니다. Azuma(http://azuma.pc.to)

## 사이트의 특이사항

Back-end : PHP 7.2
Front-end : HTML, CSS(Framework: Bootstrap 4), Javascript
DB : MySQL 5.7
Vision Manage Tool : Github
