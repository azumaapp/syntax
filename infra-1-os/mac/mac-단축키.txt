<기본>
전체 스크린샷 : 커맨드 쉬프트 3(커맨트 c 필요)
부분 스크린샷 : 커맨드 쉬프트 4(커맨트 c 필요)
잘라내기 : 커맨드 c 한 다음, 붙여넣기할 때 커맨드 옵션 v
바탕화면 보기 : Fn + F11
스플릿뷰 : 화면 넓히는 초록 버튼에 마우스 대고 있다가, 화면 왼쪽/오른쪽으로 이동 버튼 나오면 누르기(안 뜨면 옵션키 눌러보기)
맥북 바탕화면 이동 : F3키 누르고 원하는 화면 클릭
vs코드, 구글스프레드시트 등의 툴 확대/축소 : 커맨드 + 쉬프트 + - / 커맨트 + -
기호 입력 : 컨트롤 + 커맨드 + 스페이스
한자 입력 : 옵션 + 엔터 (기호는 안됨)
앱 닫기 : cmd + q
창 닫기 : cmd + w 
윈도우키 : 트랙패드 네손가락으로 오므리기 (단축키 없음)
Del키 : Fn + Delete
스프레드시트 매크로용 shift + ctrl + alt + 숫자키 : shift + opt + cmd + 숫자키

<git 관리>
terminal이나 vscode로 git을 관리한다.

<CMD>
cmd + space 후 terminal : 명령어 창 열기

<code .>

* 만일 github 등으로 당겨온 앱이라면 npm i를 통해서 vue.cmd 웹빌드를 해야한다.
