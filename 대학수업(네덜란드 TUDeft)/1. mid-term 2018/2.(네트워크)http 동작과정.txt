http 동작과정

네트워크 5계층
    애플리케이션 레이어 : message 패킷 => HTTP, SMTP, FPT (google.com을 사용자가 주소창에 입력)
        두 host는 HTTP 메시지 포멧을 통해서 통신합니다.    
    트랜스포트 레이어 : segment(출발지 포트 번호, 목적지 포트 번호) 패킷 => TCP
        신뢰성,흐름제어, 혼잡제어의 기능을 제공합니다.
    네트워크 레이어 : datagram 패킷 => IP
        segment를 datagram으로 나눈 뒤 routing합니다.
    링크 레이어 : frame 패킷 => 
        통신 링크는 경로상의 다음 노드에 전달합니다.
    물리 레이어 : bit 패킷 => 
        링크를 이루는 물질은 bit를 전송하는 역할을 맡습니다.
