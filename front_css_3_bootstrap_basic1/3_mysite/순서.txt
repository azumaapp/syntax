홈페이지 만들기

1. 공통 부분 만들기
Boostrap을 검색하고 다운받는다.
폴더 안에 압축을 해제한다.
img/core 안에 favicon.png를 추가한다.
index.html과 favicon을 연결한다.
index.html을 만들고 bootstrap.css와 .js를 연결한다.
style.css와 main.js도 각각 만들어서 index.html과 연결한다.
header와 footer 태그를 만든다.
Bootstrap API 문서를 연다. (API : Application Programming Interface)
Navbar를 참조해서 메뉴를 만든다.
각 링크의 href 프로퍼티에 html들을 연결해준다.
Footer를 검색, 참조해서 푸터를 만든다.

2. 컨텐츠 만들기 준비
완성된 index.html을 카피페해서 다른 html들을 만든다.
컨텐츠마다, section 태그로 구분한다.
style.css에서, section에 패딩, 마진을 준다.

3. 컨텐츠 만들기
<!-- ##### Content Start ##### -->
<section class="blog">
    <div class="container">
      <div class="row">
        <h1>Blog</h1>
        *** API Doc를 참조하여, 원하는 Components를 복붙 ***
      </div>
    </div>
</section>
<!-- ##### Content End ##### -->

4. 필요한 JS를 넣는다. (main.js 참조)
